# Заголовки и хуки — адаптация для @baristasss

> Формулы хуков из методики Галсона, адаптированные под стиль Кирилла Попова.

---

## Принцип

**Заголовок = обещание.** Одна мысль, одно обещание.  
Хук — это первые 1-2 строки, видные в превью Telegram без раскрытия поста.

**80% успеха поста — хук.** Без него контент не откроют.

---

## Формулы заголовков (адаптированные под @baristasss)

### 1. Провокация / Парадокс
```
[Утверждение, которое кажется неверным]
```
Примеры:
- «Telegram умирает»
- «Чем меньше правил — тем меньше аварий»
- «Реклама не работает»

### 2. Личный момент в моменте
```
[Конкретная ситуация прямо сейчас — без вступления]
```
Примеры:
- «Ночью чуть сердце не выскочило»
- «Сегодня я пробежал свой третий марафон»
- «Прочитал новости — вчера ебанули по иранской нефтянке»

### 3. Вопрос-провокация
```
[Вопрос который задаёт читатель сам себе — и хочет узнать ответ]
```
Примеры:
- «А ты токсик или душнила?»
- «Вы сейчас ахуеете»

### 4. Нумерованный инсайт
```
[Число] + [конкретное обещание]
```
Примеры:
- «5 лет я не выходил на старт»
- «3 факта которые мне доказали»

### 5. Фреймворк/термин
```
[Новое понятие которое сразу объясняет суть]
```
Примеры:
- «Шринкфляция»
- «Think Different → Think Big → Think Small»
- «Продукт Шрёдингера»

### 6. Противопоставление
```
[Как принято] vs [Как на самом деле]
```
Примеры:
- «Очевидное невероятное»
- «Правота или истина»

---

## Чеклист хука

- [ ] Понятен за 2 секунды?
- [ ] Создаёт вопрос, а не даёт ответ?
- [ ] Обращается к конкретной боли/ситуации ЦА?
- [ ] Не раскрывает суть поста до конца?
- [ ] Написан на «ты»?
- [ ] НЕ содержит канцеляризмов или клише?

---

## Специфика Telegram

- Длина хука: до 80 символов — видна в превью без раскрытия
- Первое предложение = **жирный текст** (44% постов Кирилла)
- Иногда хук = заголовок поста (отдельная строка), иногда = первое предложение

---

## Усилители

| Приём | Пример |
|-------|--------|
| Мат для точности | «сформулировал хуйню — получил хуйню» |
| Цифра из личного опыта | «4500+ сотрудников», «20 лет» |
| Самоирония | «ебанько тут я, а не система» |
| Неожиданный угол | «бухать от этого меньше не перестанут» |

---

## Антиприёмы

❌ «Сегодня я хочу поговорить о...» — нельзя  
❌ «Уважаемые читатели» — нельзя  
❌ Кликбейт без содержания — нельзя  
❌ Абстрактный заголовок («Важные мысли о бизнесе») — нельзя
