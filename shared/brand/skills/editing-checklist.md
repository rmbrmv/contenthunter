# Чек-лист редактуры — адаптация для @baristasss

> Инфостиль Ильяхова + убеждающий текст (Панда), адаптированные под стиль Кирилла Попова.

---

## Проход 1: Смысл и структура

- [ ] Понятно о чём пост за 5 секунд?
- [ ] Есть главная мысль? Она раскрыта?
- [ ] Каждый абзац двигает к этой мысли?
- [ ] Нет повторов одного и того же разными словами?
- [ ] Первое предложение — хук? Создаёт желание читать дальше?
- [ ] Финал — удар или неожиданность? Не расплылся?

---

## Проход 2: Стиль Кирилла

- [ ] Текст написан на «ты»?
- [ ] Нет обращения «Вы» (заглавная)?
- [ ] Живой язык — как разговор за столом?
- [ ] Нет канцелярита? («в рамках данного», «осуществить реализацию»)
- [ ] Нет пустых прилагательных? («уникальный», «эффективный», «качественный»)
- [ ] Конкретика вместо абстракций? (цифры, имена, детали)
- [ ] Нет лишних вводных слов? («в принципе», «как говорится», «так сказать»)

---

## Проход 3: Форматирование

- [ ] Первое предложение — жирным (`**текст**`)?
- [ ] Абзацы короткие (1-3 предложения)?
- [ ] Воздух между смысловыми блоками?
- [ ] Списки нумерованные где есть порядок, тире где нет?
- [ ] Эмодзи не больше 2-3? Только из набора: 👉❤🔥✅💬👇📌?
- [ ] CTA отсутствует (если не указан в задании)?

---

## Проход 4: Инфостиль

- [ ] Убраны слова-паразиты: «очень», «действительно», «буквально»?
- [ ] Оценки заменены фактами? («хороший результат» → «конверсия выросла с 1% до 3%»)
- [ ] Предложения простые? Нет причастных оборотов на 3 строки?
- [ ] Текст читается вслух без запинок?
- [ ] Сокращено в 1.5 раза от черновика?

---

## Проход 5: Голос Кирилла

- [ ] Есть личный опыт или конкретный кейс?
- [ ] Позиция однозначная — нет «с одной стороны/с другой стороны»?
- [ ] Финальная строка — короткая и сильная?
- [ ] Пост можно переслать другу?
- [ ] Читатель узнаёт себя в посте?

---

## Стоп-слова для @baristasss

Заменить или удалить:

| Нельзя | Можно |
|--------|-------|
| «Уникальный» | Конкретное отличие |
| «Качественный» | Что именно качественно |
| «Эффективный» | Насколько, в чём |
| «Важно отметить» | Просто отмечай |
| «Следует сказать» | Просто скажи |
| «В контексте» | Убери совсем |
| «Осуществить» | Сделать |
| «Данный» | Этот |
| «Является» | Есть / = |
| «Приоритетный» | Главный |
