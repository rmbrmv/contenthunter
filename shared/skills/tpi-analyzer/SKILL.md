# TPI Analyzer

Система оценки сотрудников по 5 метрикам. Оценка выставляется AI, акцептируется руководителем.

## Структура оценки

**Блок 1 — Индекс командной надёжности (ИКН × 0.45)**
- Коммуникация × 0.40
- Дисциплина × 0.35
- Вовлечённость × 0.25

**Блок 2 — Индекс результативности (ИР × 0.55)**
- Экспертность × 0.45
- Результативность × 0.55

**TPI = ИКН×0.45 + ИР×0.55** → шкала 0–10

## Сценарии использования

### 1. Еженедельная оценка сотрудника

```bash
python3 scripts/collect_data.py --employee "Ежков Олег" --week 2026-03-02
```

Скрипт соберёт данные из БД за указанную неделю и выведет заполненный промпт.
Подай этот промпт в модель с системным промптом из `references/master-prompt.md`.
Результат (JSON) сохрани через:

```bash
python3 scripts/save_tpi.py --json '...'
```

### 2. Оценка всей команды за неделю

```bash
python3 scripts/collect_data.py --all --week 2026-03-02
```

Скрипт пройдёт по всем активным сотрудникам из `team.employees`.
После прогона каждого через модель — сформируй дашборд промптом из `references/dashboard-prompt.md`.

### 3. Ежедневный экстренный мониторинг

```bash
python3 scripts/daily_check.py --date 2026-03-05
```

Собирает активность за день. Подай в модель с промптом из `references/emergency-prompt.md`.
Если ответ не "OK" — отправь руководителю.

## БД и схема

- `team.employees` — 27 сотрудников с telegram_username
- `public.telegram_messages` — сообщения (поле `username` = telegram_username без @)
- `mymeet.meetings` — Zoom-транскрипты
- `public.people_ratings` — куда сохраняются итоговые оценки

Подробная схема: `references/db-schema.md`

## Предиктор

Система автоматически выставляет флаги на основе тренда TPI:
- `RISK_OF_LEAVING` — TPI падает 3 недели + вовлечённость < 4
- `GROWTH_CANDIDATE` — TPI растёт 3 недели + вовлечённость > 7
- `DECLINING` — TPI падает 2 недели
- `IMPROVING` — TPI растёт 2 недели
- `STABLE` — нет тренда

## Командный TPI

Средний TPI по отделам считается автоматически при прогоне всей команды. Позволяет сравнивать отделы между собой.

## Интерпретация TPI

| TPI | Статус | Действие |
|-----|--------|----------|
| 9–10 | 🟢 Звезда | Отметить публично |
| 7–8 | 🟢 Сильный | Давать больше ответственности |
| 5–6 | 🟡 Норма | Наблюдать |
| 3–4 | 🟠 Риск | 1-on-1 |
| 0–2 | 🔴 Кризис | Немедленное вмешательство |
