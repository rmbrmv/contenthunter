# ICP — Целевая ниша Content Hunter

**Дата:** 2026-03-05
**Источник:** Встреча Кирилл + Роман + Аня, 03.03.2026

## Критерии идеального клиента

| # | Критерий | Описание |
|---|----------|----------|
| 1 | **Продукт без привязки к ГЕО** | Продукт/услуга продаётся по всей России/СНГ, не локальный бизнес |
| 2 | **Контент** | Есть готовый контент ИЛИ возможность его делать + регулярно публикует рилсы в SMM |
| 3 | **Широкая ЦА** | Целевая аудитория не узкая ниша, массовый продукт |
| 4 | **Низкий/средний чек** | Продукт доступный, не премиум/люкс |
| 5 | **Бюджет от 150к/мес** | Готов инвестировать в маркетинг минимум 150 000 ₽ в месяц |
| 6 | **Диджитал-мышление** | Понимает ценность digital-маркетинга, есть горящий запрос на KPI и результат |

## Применение
Использовать при квалификации входящих лидов и ревизии текущей клиентской базы.
