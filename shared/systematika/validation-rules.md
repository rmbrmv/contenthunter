# Правила валидации результата

## ⚠️ ПЕРЕД ОТПРАВКОЙ РЕЗУЛЬТАТА — САМОПРОВЕРКА

После выполнения любого задания, перед отправкой ответа, проверь:

1. **Полнота** — все ли пункты задания выполнены? Ничего не пропустил?
2. **Источник** — данные взяты из конкретных файлов/источников, а не выдуманы?
3. **Цифры** — все числа проверены? Можешь указать откуда взял каждую цифру?
4. **Формат** — ответ в запрошенном формате? Если был шаблон — следуешь ему?
5. **Повторяемость** — если задание повторить, результат будет таким же?

Если на любой пункт ответ "нет" — исправь перед отправкой.
