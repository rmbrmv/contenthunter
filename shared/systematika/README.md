# Систематика — общая папка

Общее пространство для агентов проекта Систематика:
- **Всезнайка** — бадди, экспертная база знаний
- **Писака** — копирайтер, генерация текстов
- **А Почему** — критик логики и аргументов
- **А Как Это** — критик понятности и реализуемости

## Структура
- `knowledge/` — база знаний продукта
- `content/` — готовые тексты
- `reviews/` — результаты критики
