## Использование

Данные также в PostgreSQL:
- `team.employees` — 27 сотрудников с Telegram-никами
- `team.people_aliases` — 23 Zoom-алиаса → 14 сотрудников

```sql
-- Найти сотрудника по Zoom-имени
SELECT e.name, e.telegram_username FROM team.employees e
JOIN team.people_aliases pa ON pa.employee_id = e.id
WHERE pa.zoom_name ILIKE '%искомое_имя%';
```
