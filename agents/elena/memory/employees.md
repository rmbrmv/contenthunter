# Особые условия работы сотрудников

## Галым Советов

**График работы:** нестандартный
- Рабочие дни: выходные и праздничные дни по казахстанскому календарю
- Выходные дни: обычные рабочие дни (понедельник–пятница)
- Отчёты должны присылать в его рабочие дни (то есть в казахстанские выходные и праздники)
