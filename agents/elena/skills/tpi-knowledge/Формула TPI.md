# Формула Team Power Index

> Как два индекса складываются в один финальный TPI.
> Оценка выставляется еженедельно по каждому сотруднику.

---

## Структура расчёта

```
┌─────────────────────────────────────┐
│     ИНДЕКС КОМАНДНОЙ НАДЁЖНОСТИ     │
│                                     │
│  Коммуникация    × 0.40             │
│  Дисциплина      × 0.35             │
│  Вовлечённость   × 0.25             │
│                         = ИКН (0–10)│
└─────────────────────────────────────┘
              ↓ × 0.45

┌─────────────────────────────────────┐
│     ИНДЕКС РЕЗУЛЬТАТИВНОСТИ         │
│                                     │
│  Экспертность    × 0.45             │
│  Результативность× 0.55             │
│                         = ИР (0–10) │
└─────────────────────────────────────┘
              ↓ × 0.55

┌─────────────────────────────────────┐
│        TEAM POWER INDEX             │
│   TPI = ИКН×0.45 + ИР×0.55         │
│                         = TPI (0–10)│
└─────────────────────────────────────┘
```

---

## Веса и обоснование

| Метрика | Вес внутри блока | Обоснование |
|---------|-----------------|-------------|
| Коммуникация | 40% | Самый заметный сигнал деградации |
| Дисциплина | 35% | Прямая связь с надёжностью |
| Вовлечённость | 25% | Сложнее измерить, но важна |
| Экспертность | 45% | Качество работы с клиентом |
| Результативность | 55% | Прямая связь с деньгами |
| Блок надёжности | 45% | Предиктор будущего результата |
| Блок результативности | 55% | Текущий результат |

> ⚠️ Веса предварительные. Требуют калибровки после первого месяца работы с реальными данными.

## Как выставляется оценка

### Шаг 1 — AI выставляет предварительную оценку
Система анализирует данные за неделю по всем источникам и предлагает оценку по каждой метрике.

### Шаг 2 — Руководитель акцептирует или корректирует
- Видит оценку AI
- Может согласиться или изменить
- Может добавить контекст который система не видела (личный разговор, ситуация)
- Финальная оценка фиксируется

### Шаг 3 — Оценка записывается в динамику
- Сравнивается с прошлой неделей
- Формируется тренд: растёт / стабильно / падает

|-----|--------|------------|
| 9–10 | 🟢 Звезда | Отметить публично, думать о повышении |

| 5–6 | 🟡 Норма | Наблюдать за динамикой |
| 3–4 | 🟠 Риск | Запланировать 1-on-1, разобраться в причинах |
| 0–2 | 🔴 Кризис | Немедленное вмешательство |

## Привязка к деньгам

### Логика
Команда с TPI ниже 10 работает не на полную мощность. Разница — это потери.

### Формула
```
Потенциальная выручка = Текущая выручка ÷ (TPI команды ÷ 10)
Потери = Потенциальная выручка − Текущая выручка
```

### Пример
```
Текущая выручка: 1 000 000 ₽
TPI команды: 6.5

Потенциал: 1 000 000 ÷ 0.65 = 1 538 000 ₽
Потери: 538 000 ₽ в месяц
```

> Это не точный расчёт — это управленческий ориентир.
> Показывает масштаб возможного роста при той же команде.

## Еженедельный цикл

```
Пн–Пт  →  Система собирает данные
Пт–Сб  →  AI формирует предварительные оценки
Вс     →  Руководитель акцептирует оценки (15–20 мин)
Пн     →  Дашборд с новыми оценками и динамикой
```
