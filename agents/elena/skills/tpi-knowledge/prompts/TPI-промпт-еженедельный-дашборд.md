# TPI — Промпт еженедельного дашборда

> Промпт для формирования итогового отчёта по всей команде.
> Запускается каждый понедельник после расчёта TPI по каждому сотруднику.

---

## USER PROMPT

```
На основе оценок TPI за неделю сформируй еженедельный дашборд для руководителя.

## Период
{date_from} – {date_to}

## Данные по команде (JSON-массив оценок)
{team_tpi_array}

## Формат ответа — строго Markdown

# 📊 TPI-дашборд — {date_from}–{date_to}

## Рейтинг команды

| # | Сотрудник | TPI | ИКН | ИР | Динамика | Статус |
|---|-----------|-----|-----|----|----------|--------|
| 1 | ... | ... | ... | ... | ↑/↓/= | 🟢/🟡/🟠/🔴 |

(сортировка по TPI убыванию)

Статусы:
🟢 7+ — норма/хорошо
🟡 5–7 — наблюдение
🟠 3–5 — риск
🔴 <3 — кризис
```
