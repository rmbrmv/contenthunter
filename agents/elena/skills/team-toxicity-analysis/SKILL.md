# Team Toxicity Analysis Skill

Анализ тональности и токсичности командной коммуникации.

## Описание

Анализирует корпоративные чаты и внутренние встречи команды за указанный период, выявляет уровень токсичности по каждому сотруднику и выдаёт отчёт.

## Использование

```bash
python3 analyze_toxicity.py --days 7 --format table
```

## Параметры

- `--days`: Количество дней для анализа (по умолчанию 7)
- `--format`: Формат вывода: `table` или `detailed`

## Что анализируется

**Источники:**
- Корпоративные чаты (Telegram)
- Внутренние встречи (Zoom транскрипты)

**Метрики:**
- Резкие выражения в чатах: `дебил, блин, задолб, блядь, ебать, пиздец, хуйня`
- Матерные слова на встречах: `пиздец, хуйня, блядь, ебать`
- Направленность негатива (на людей vs на процесс)

**Формула:**
- Чаты: (резкие маркеры / сообщения) × 20
- Встречи: (матерные слова / высказывания) × 25
- Итог: максимум между чатом и встречами

## Оценки

- **0/10** — Отлично (полностью профессионален)
- **0.1–2/10** — Хорошо (норма для компании)
- **2.1+/10** — Внимание (требуется обсуждение)

## Вывод

Таблица по каждому сотруднику + общие выводы:
- Количество нетоксичных / хороших / проблемных
- Статус команды в целом
- Рекомендации

## Файлы

- `analyze_toxicity.py` — основной скрипт
- `SKILL.md` — это документация
- `example_output.txt` — пример отчёта

## Интеграция

Используется в еженедельном крон-запросе на оценку команды (каждый понедельник 18:00 МСК).

Запускается перед запросом баллов, чтобы дать контекст лидеру.
