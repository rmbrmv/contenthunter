# Approved Patterns

## 2026-03-22 — Ежедневный итог питания в HTML

**Context:** Конец дня, Роман просит подвести итог по питанию и воде.

**Pattern:** Генерировать HTML-отчёт с:
- Сводка КБЖУ за день (факт vs цель)
- Прогресс-бары калорий и воды
- Все приёмы пищи с таблицами Б/Ж/У/ккал
- Водный баланс с разбивкой по источникам
- Оценка дня: что хорошо / что улучшить

**Файл:** `projects/roman/daily/YYYY-MM-DD.html`
**Отправка:** message tool → filePath → topic:5 в группе Здоровье

**Cron:** 21:30 МСК (18:30 UTC) ежедневно — автоматический итог

**Шаблон:** `/root/.openclaw/workspace-nutritionist/projects/roman/daily/2026-03-22.html` — использовать как образец стиля.
