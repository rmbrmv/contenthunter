## 2026-03-23: Пропуск memory/2026-03-21.md
**Что не сработало:** Файл за 21 марта отсутствует — день выпал из истории.
**Почему:** Heartbeat, судя по всему, не отработал или результат не был записан в файл.
**Урок:** Пропуск heartbeat не катастрофа, но фиксировать факт пропуска в следующей рефлексии — обязательно. Память не должна иметь незамеченных дыр.

## 2026-03-15: Отсутствие memory-файлов за 13–15 марта
**Что не сработало:** За 3 дня активной работы нет ни одного memory/YYYY-MM-DD.md. События и решения не зафиксированы.
**Почему:** Не создавались файлы по итогам сеансов. "Умственные заметки" не выживают между сессиями.
**Урок:** В конце каждого рабочего сеанса создавать memory/YYYY-MM-DD.md. Даже 3–5 строк лучше, чем ничего.

## 2026-03-17: Системный сбой — 3 дня без рабочих memory-файлов
**Что не сработало:** Три дня подряд (15, 16, 17 марта) нет ни одного рабочего memory-файла. Только heartbeat-рефлексии. Проблема не единичная — это системный сбой.
**Почему:** У этого агента нет рабочих задач от Романа в эти дни, либо сеансы не инициируются вовсе. Heartbeat — единственная точка активности. Ритуал "открыть memory в начале сеанса" невозможен, если сеанса нет.
**Урок:** Разделить два случая: (1) нет задач → это нормально, фиксировать в heartbeat "нет активности"; (2) есть задачи → memory обязателен. Не путать "нет файла" с "нет работы".

## 2026-03-16: Паттерн "нет памяти" повторяется второй день подряд
**Что не сработало:** Вчера зафиксировал антипаттерн "не создаю memory-файлы" — сегодня снова нет файла за 16 марта. Урок не применился.
**Почему:** Осознание проблемы есть только в момент рефлексии (heartbeat), а не в момент работы. Нет триггера "закончил сеанс → создай файл".
**Урок:** Нужен жёсткий ритуал: первое действие в сеансе — открыть/создать memory/YYYY-MM-DD.md. Последнее — дописать итог. Без этого цикл не замкнётся.

## 2026-03-15: Пустые learning-файлы при наличии активности
**Что не сработало:** approved-patterns.md и anti-patterns.md оставались пустыми несмотря на онбординг и (предположительно) задачи в марте.
**Почему:** Рефлексия не была встроена в рабочий процесс — только в heartbeat.
**Урок:** После любой завершённой задачи фиксировать урок в learning/ сразу, не откладывая до heartbeat.
