## 🏆 ТОП-10 Виртуальных Офисов

### 1. **Gather.town** ⭐⭐⭐⭐⭐
🔗 https://gather.town  
📸 Скриншоты: нет (сайт на Framer, заблокированы)

**Дизайн:**
- **Стиль:** 2D пиксель-арт, изометрический вид сверху (top-down)
- **Аватары:** 8-bit спрайты, кастомизируемые, двигаются по карте W/A/S/D
- **Комнаты:** Опен-спейс, конференц-залы, переговорки, кофейни, лаунж-зоны
- **Proximity chat:** Видео/аудио включается автоматически когда аватары близко
- **Статусы:** онлайн/оффлайн/занят через аватар
- **Карта:** Tile-based, настраиваемые офисные планы этажей

**Ключевые фишки для AI-офиса:**
- Proximity-based взаимодействие — агенты "разговаривают" когда близко
- Разные зоны = разные контексты (whiteboard room, meeting room)
- Avatars с именами и статусами над головой
- Карта видна целиком (bird's eye view)

### 2. **SkyOffice** ⭐⭐⭐⭐⭐ (Open Source!)
🔗 https://github.com/kevinshen56714/SkyOffice  
🌐 https://skyoffice.netlify.app  
📸 `office-screenshots/skyoffice-main.png`, `skyoffice-room.png`, `skyoffice-concept.jpg`

**Дизайн:**
- **Стиль:** 2D пиксель-арт top-down, аналог Gather.town
- **Аватары:** RPG-style персонажи с именами
- **Комнаты:** Lobby, Conference rooms, Private rooms, Whiteboard rooms
- **Диалоговые пузыри:** Text bubbles над аватарами в реальном времени
- **Экраны:** Встроенный whiteboard (WBO), screen sharing
- **Stack:** Phaser3 + React + Colyseus (WebSocket) + PeerJS (WebRTC)

**Что видно на скриншотах:**
- Офис вид сверху, тайловая карта с мебелью (столы, стулья, растения, ковры)
- Аватары двигаются по карте, над каждым — имя и speech bubble
- Конференц-зал с экраном для видео-чата (встроен в тайл)
- Разные текстуры полов для разных зон (ковёр = переговорная)

**⚡ Это лучший open-source референс для нас!**

### 3. **Sococo** ⭐⭐⭐⭐
🔗 https://sococo.com  
📸 (нет прямого скриншота, CDN заблокирован)

**Дизайн:**
- **Стиль:** Flat 2D, вид сверху (план этажа), НЕ пиксельный
- **Аватары:** Круглые аватары/фотографии в реальном времени с видео-превью
- **Комнаты:** Реалистичный план офиса: reception, open office, meeting rooms, hallways
- **Статусы:** Зелёный (доступен), красный (занят), серый (оффлайн) — видно на карте
- **Особенность:** Когда входишь в комнату — автоматически начинается разговор

**Ключевые фишки:**
- Комнаты как отдельные пространства с дверьми
- Видеопревью прямо в "окне" аватара
- Видно кто в какой комнате находится без входа
- Очень "офисный" дизайн — прямо как план здания

## 🎨 Ключевые дизайн-паттерны для AI-офиса

### Пространство
- **Tile-based карта** (16×16 или 32×32 тайлы) — стандарт для 2D офисов
- **Bird's eye view** — вид сверху, видно всю карту
- **Зонирование:** разные текстуры/цвета для разных зон (ковёр = переговорная)
- **Мебель:** столы, стулья, растения, доски — создают ощущение офиса

### Аватары (для AI-агентов)
- **Имя над головой** — всегда видно кто есть кто
- **Статус-индикатор** — цветная точка (зелёный/красный/серый)
- **Speech bubble** — что агент сейчас "говорит"/думает
- **Анимация движения** — spritesheet walking animation (4 направления)

### Взаимодействие
- **Proximity zones** — агенты взаимодействуют когда "рядом"
- **Room entry** — вход в комнату = вход в контекст
- **Tap/click** — клик на агента = начало взаимодействия
- **Статусы занятости** — видно кто свободен/занят/оффлайн

### UI-элементы
- **Sidebar** — список присутствующих с мини-аватарами
- **Chat overlay** — чат поверх карты
- **Minimap** — мини-карта в углу для навигации

## 💡 Рекомендации для AI-агентного офиса

**Лучший старт:** Взять SkyOffice как базу (open source, Phaser3 + React)

**Визуальная концепция:**
1. Pixel art 2D top-down офис (32×32 тайлы)
2. Каждый AI-агент = уникальный пиксельный аватар со своим цветом/образом
3. Комнаты = рабочие контексты (Content Room, Code Room, Analytics Room, etc.)
4. Speech bubbles = текущая задача агента
5. Статус = idle/working/thinking/blocked
6. Bird's eye view + возможность зайти в комнату

**Технический стек:**
- Phaser3 — рендеринг карты и аватаров
- Colyseus — синхронизация состояний
- React — UI overlays
- Socket.IO — real-time обновления статусов
