### Рекомендации:
1. **Обновиться до v2026.3.8** — там 3 security фикса включая path traversal в скиллах. Критично для нашей фермы из 18 агентов
2. **Проверить наши скиллы через RankClaw** (rankclaw.com) — 7.5% malicious по отрасли, надо убедиться что наши чистые
3. **ACP Provenance** (`openclaw acp --provenance meta+receipt`) — новая фича для трейсинга multi-agent цепочек. Полезно для дебага нашей команды
4. **Brave LLM Context mode** — включить `tools.web.search.brave.mode: "llm-context"` для лучших веб-результатов с metadata
5. **talk.silenceTimeoutMs** — настроить тайм-аут паузы в Talk mode если используется
6. **Telegram дублирование** — если замечали двойные ответы, v2026.3.8 фиксит #40005. Если баг сохраняется — удалить `~/.openclaw/telegram/update-offset-default.json`
7. **autocouncil** (github.com/shadmau/autocouncil) — легковесный CLI для self-review агентских планов перед выполнением (PASS/REVISE/BLOCK паттерн). Стоит рассмотреть для критичных операций
