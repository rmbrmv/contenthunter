### v2026.3.22 (вышел 2026-03-23, опубликован 2026-03-23 11:11 UTC)
- URL: https://github.com/openclaw/openclaw/releases/tag/v2026.3.22
- Автор: steipete

**Breaking changes:**
- Plugins/install: bare `openclaw plugins install <package>` теперь предпочитает ClawHub перед npm
- Browser/Chrome MCP: удалён legacy Chrome extension relay path, bundled extension assets, `driver: "extension"`, `browser.relayBindHost`. Нужно запустить `openclaw doctor --fix`
- Tools/image generation: стандартизован путь на `image_generate` tool; удалён `nano-banana-pro` skill
- Skills/image generation: удалён bundled `nano-banana-pro` skill; использовать `agents.defaults.imageGenerationModel.primary: "google/gemini-3-pro-image-preview"`
- Plugins/SDK: новая поверхность `openclaw/plugin-sdk/*`; `openclaw/extension-api` удалён
- Config/env: удалены legacy `CLAWDBOT_*` и `MOLTBOT_*` env names; использовать `OPENCLAW_*`
- Config/state: удалён `.moltbot` state-dir и `moltbot.json` auto-detection

**Ключевые новые фичи:**
- ClawHub/install: native `openclaw skills search|install|update` flows + `openclaw plugins install clawhub:<package>`
- Plugins/marketplaces: Claude marketplace registry resolution, `plugin@marketplace` installs
- Install/update: GitHub `main` через `openclaw update --tag main`
- Plugins/bundles: совместимость Codex, Claude, Cursor bundle discovery/install
- Models/OpenAI: default setup model → `openai/gpt-5.4`; Codex → `openai-codex/gpt-5.4`
- Commands/btw: `/btw` side questions без изменения контекста сессии
- Sandbox/runtime: pluggable sandbox backends, OpenShell backend с `mirror` и `remote` workspace modes
- Sandbox/SSH: core SSH sandbox backend
- Browser/existing-session: поддержка `browser.profiles.<name>.userDataDir` для Brave, Edge и других Chromium-based
- Models/Anthropic Vertex: core `anthropic-vertex` provider для Claude via Google Vertex AI
- Web tools/Exa: bundled Exa web-search plugin
- Web tools/Tavily: bundled Tavily web-search с `tavily_search` и `tavily_extract` tools
- Web tools/Firecrawl: bundled Firecrawl plugin с `firecrawl_search` и `firecrawl_scrape`
- Models/OpenAI: `gpt-5.4-mini` и `gpt-5.4-nano` forward-compat support
- xAI/models: sync Grok catalog к текущим Pi-backed IDs
- Plugins/MiniMax: MiniMax-M2.7 и MiniMax-M2.7-highspeed; default модель M2.5 → M2.7
- Telegram/topics: auto-rename DM forum topics с LLM-generated labels
- Telegram/apiRoot: per-account custom Bot API endpoint
- Feishu/streaming: `onReasoningStream` и `onReasoningEnd` для streaming cards
- Android/Talk: Talk speech synthesis через gateway `talk.speak`
- Android/nodes: `callLog.search` и `sms.search`
- Control UI/chat: expand-to-canvas button на assistant chat bubbles
- Plugins/Matrix: новый Matrix plugin на `matrix-js-sdk`

**Security fixes:**
- Media/Windows security: блокировка remote-host `file://` media URLs и UNC/network paths
- Gateway/discovery: fail closed на unresolved Bonjour/DNS-SD endpoints
- Security/pairing: binding iOS setup codes к intended node profile
- Nostr/security: enforce inbound DM policy перед decrypt
- Synology Chat/security: reply delivery bound к stable numeric `user_id`
