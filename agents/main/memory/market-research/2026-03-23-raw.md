#### BREAKING CHANGES (key ones):
1. **Plugins/install**: `openclaw plugins install <package>` теперь предпочитает ClawHub перед npm
2. **Browser/Chrome MCP**: удалён legacy Chrome extension relay path, `driver: "extension"` и `browser.relayBindHost`. Нужно запустить `openclaw doctor --fix` для миграции
3. **Tools/image generation**: стандартизирован путь на core `image_generate` tool. Старый `nano-banana-pro` удалён — использовать `agents.defaults.imageGenerationModel`
4. **Skills/image generation**: удалён bundled `nano-banana-pro` skill. Использовать `agents.defaults.imageGenerationModel.primary: "google/gemini-3-pro-image-preview"`
5. **Plugins/SDK**: новый SDK surface `openclaw/plugin-sdk/*`; `openclaw/extension-api` удалён без шима
6. **Config/env**: удалены legacy `CLAWDBOT_*` и `MOLTBOT_*` env names — использовать `OPENCLAW_*`
7. **Config/state**: удалён legacy `.moltbot` state-dir — перенести в `~/.openclaw`
8. **Exec/env sandbox**: блокировка JVM injection vars (`MAVEN_OPTS`, `SBT_OPTS`, `GRADLE_OPTS`, `ANT_OPTS`), glibc tunables, .NET deps
9. **Discord/commands**: Carbon reconcile по умолчанию
10. **Voice-call/webhooks**: reject missing provider signature headers, drop pre-auth body budget до 64KB/5s

#### KEY SECURITY FIXES:
- Media/Windows security: блокировка remote-host `file://` URLs и UNC paths (SMB credential leak)
- Gateway/discovery: fail closed на unresolved Bonjour/DNS-SD endpoints
- Security/pairing: bind iOS setup codes to intended node profile
- Nostr/security: enforce inbound DM policy before decrypt, rate/size guards
- Synology Chat/security: reply delivery bound к `user_id`, не username
- Security/exec: блокировка `jq` из safe-bin allowlist (env builtin мог дампить secrets)
- Security/exec approvals: escape Hangul filler Unicode code points в approval prompts
- Security/network: SSRF pinning hardening для explicit-proxy

#### KEY FIXES:
- Agents/default timeout: 600s → 48h
- Gateway/startup: load bundled channel plugins из compiled `dist/extensions` (WhatsApp cold start: seconds vs minutes)
- CLI/startup: lazy-load channels для экономии RAM
- Agents/openai-compatible tool calls: дедупликация repeated tool call ids (HTTP 400 fix)
- Control UI/session routing: preserve external delivery routes в webchat sessions
- Telegram/replies: `allow_sending_without_reply` для deleted parent messages
- Android/location: fix crash `Already resumed`
- WhatsApp/reconnect: fix append recency filter, protobuf Long timestamps
- Build/memory tools: `dist/cli/memory-cli.js` как stable entry (не hashed bundle)
