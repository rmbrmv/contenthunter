### ТОП 2: "OpenClaw Lobster Deep Dive" — OpenClaw_NF
- **ID:** 1EsQa1RH3RM
- **Views:** 1793 | **Likes:** 43 | **Comments:** 2 | **ER:** 2.51%
- **URL:** https://youtube.com/watch?v=1EsQa1RH3RM
- **Ключевые инсайты из транскрипта:**
  - **Lobster** — нативный workflow engine OpenClaw, выпущен в феврале 2026
  - Решает 3 проблемы LLM-оркестрации: дорого (каждый шаг тратит токены), недетерминистично (разный результат на одинаковом input), ненадёжно (20-шаговый pipeline имеет 33% шанс провала)
  - Архитектура: "small CLI + JS pipes + approvals" — каждый шаг это CLI-команда, берущая JSON на вход и выдающая JSON
  - LLM участвует ТОЛЬКО в тех шагах где нужно reasoning, оркестрация полностью вне LLM
  - Approval gates: halt execution перед side-effect'ами
  - Результат: детерминистичное поведение, fraction токенов, аудируемый execution log
  - **Для enterprise**: compliance team может подписать execution log

### 1EsQa1RH3RM — "OpenClaw Lobster Deep Dive"
Подробный технический разбор Lobster — нативного workflow engine. Три проблемы LLM-оркестрации: стоимость (каждый переход тратит токены), недетерминизм, ненадёжность (33% failure rate на 20-шаговых pipelines). Lobster решает: typed data structure pipeline, JSON piped между шагами, approval gates, LLM только для reasoning-шагов. Архитектура "small CLI + JS pipes + approvals". Enterprise-ready: аудируемый execution log. Результат: идентичное поведение при каждом запуске, fraction стоимости, compliance-friendly.

### GwSYhTrWWuA — "5 AI Agents, 1 Discord Server"

Heartbeat заставляет агентов проверять taskboard каждые 30-60 минут. Агенты сами думают как продвинуть проект 3-4 раза в день. Каждый department head имеет свои sub-agents.

### RaIaU7irD9w — "Brainstorm Better with Knowledge Graphs"
