### Ключевые изменения v2026.3.2 (СЕГОДНЯ):

**NEW FEATURES:**
- 🔐 **Secrets/SecretRef** — расширенная поддержка SecretsRef по всем 64 поверхностям учётных данных. Нераскрытые ссылки сразу фейлят активные поверхности (fail fast)
- 📄 **Tools/PDF analysis** — первоклассный PDF-инструмент с нативной поддержкой Anthropic и Google PDF. Настройки: `agents.defaults.pdfModel`, `pdfMaxBytesMb`, `pdfMaxPages`
- 📎 **Sessions/Attachments** — встроенная поддержка вложений файлов в `sessions_spawn` (только subagent runtime), base64/utf8 кодировка, лимиты через `tools.sessions_spawn.attachments`
- 🤖 **Models/MiniMax** — первоклассная поддержка MiniMax-M2.5-highspeed
- 📱 **Telegram/Streaming defaults** — `channels.telegram.streaming` теперь **по умолчанию `partial`** (было `off`) — новые установки сразу получают live preview
- 🧠 **Memory/Ollama embeddings** — `memorySearch.provider = "ollama"` и `memorySearch.fallback = "ollama"` — локальные embeddings через Ollama
- 🔌 **Plugin SDK/channel extensibility** — `channelRuntime` на `ChannelGatewayContext` для внешних плагинов
- 🌐 **Zalo Personal plugin** — перестроен на нативный `zca-js` (без внешних CLI бинарников)
- ⚡ **Plugin runtime/STT** — `api.runtime.stt.transcribeAudioFile(...)` для расшифровки аудио из расширений
- 🔔 **Media/audio echo** — `tools.media.audio.echoTranscript` — отправка транскрипта перед ответом агента
- 🎭 **CLI/Banner taglines** — `cli.banner.taglineMode (random | default | off)`
- 🔔 **Plugin runtime/system** — `runtime.system.requestHeartbeatNow(...)` — принудительный heartbeat из расширений

**BREAKING CHANGES:**
- ⚠️ **tools.profile** теперь по умолчанию `messaging` для новых установок (не `coding/system`)
- ⚠️ **ACP dispatch** теперь включён по умолчанию (отключить: `acp.dispatch.enabled=false`)
- ⚠️ **Plugin SDK** — `api.registerHttpHandler()` удалён → только `api.registerHttpRoute()`
- ⚠️ **Zalo Personal** — больше не нужны внешние CLI (`openzca`, `zca-cli`), нужен `openclaw channels login --channel zalouser`

**KEY FIXES:**
- Telegram/Streaming DM: `sendMessageDraft` для private preview

- Feishu: множество фиксов (routing, topic, dedup, broadcast)
- Security: SSRF guard, webhook auth hardening, gateway canonicalization
- Sessions/Lock recovery: корректное определение recycled Linux PIDs
- Cron: HEARTBEAT_OK leak в изолированных сессиях исправлен
