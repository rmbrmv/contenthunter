## 🏆 ТОП-5 фишек для ContentHunter

### 1. Claw Router — экономия 78% на API
**Источник:** YouTube (agnDZQqFKIk)
**Суть:** Скилл, который автоматически маршрутизирует задачи к разным LLM в зависимости от сложности. Простые запросы → Haiku, средние → Sonnet, сложные → Opus.

**Что нам даст:**
- Автоматическая экономия токенов без ручного распределения моделей
- Генри уже делает Model Router — посмотреть как Claw Router реализован, перенять идеи
- Потенциально заменить ручное распределение моделей (сейчас в MEMORY.md)

**Как внедрить:** Делегировать Генри — сравнить с его model-router, возможно интегрировать подход.

**Приоритет:** 🔴 Высокий (экономия $$)

### 2. hmem-MCP — иерархическая память вместо раздувания контекста
**Источник:** HN (github.com/Bumblebiber/hmem)
**Суть:** MCP-сервер с иерархической памятью — агент сам решает что загружать по тегам, вместо загрузки всех memory файлов в контекст.

**Что нам даст:**
- У нас проблема: Фёдор забился до 174K/200K, пришлось делать /reset
- hmem загружает только нужные блоки памяти по тегам — контекст не раздувается
- Агенты перестанут "забывать" — память не теряется при /reset

**Как внедрить:** Изучить hmem, протестировать на одном агенте (Фёдор — первый кандидат). Если работает — раскатать на всех.

**Приоритет:** 🔴 Высокий (решает реальную боль)

---

### 3. DiffMem — git-based память (790+ stars, в продакшене)
**Источник:** HN (github.com/Growth-Kinetics/DiffMem)
**Суть:** Память агентов через git-коммиты вместо .md файлов. 4 режима: basic/wide/deep/temporal. Multi-tenancy через git worktrees.

**Что нам даст:**
- История всех изменений памяти (git log) — можно откатить
- Temporal режим — агент помнит КОГДА что произошло
- 790+ stars = зрелый проект, уже в production

**Как внедрить:** Альтернатива hmem. Оценить что лучше подходит. Делегировать Генри на R&D.

**Приоритет:** 🟡 Средний (R&D, не срочно)

### 4. OpenClaw v2026.3.2 — PDF инструмент + Ollama embeddings
**Источник:** GitHub Releases (сегодня)
**Суть:** Крупный релиз. Ключевые для нас фичи:
- **PDF tool** — нативный анализ PDF (контракты, брифы, документы клиентов)
- **Ollama embeddings** — локальные embeddings для memory_search (бесплатно, без Gemini API)
- **Telegram streaming по умолчанию** — live preview уже включён
- **Attachments в sessions_spawn** — можно передавать файлы субагентам

**Что нам даст:**
- PDF-анализ клиентских документов прямо в чате
- Переход на локальные embeddings = экономия на Gemini
- Быстрее ответы в Telegram (streaming)

**Как внедрить:** Обновить OpenClaw до v2026.3.2. Внимание на breaking changes: `tools.profile` по умолчанию `messaging`.

**Приоритет:** 🔴 Высокий (обновление + новые возможности)
