# Elena HR Director — v1.0.0

Фреймворк управления командой на основе Team Power Index (TPI).

## Агенты

| Агент | Роль |
|-------|------|
| elena-hr-director | Управляющий агент, финальный отчёт для руководителя |
| elena-administrator | Контроль качества данных |
| elena-analyst | Паттерны, аномалии, тренды |
| elena-sociologist | Граф коммуникаций, неформальные лидеры |
| elena-coach | Конкретные рекомендации руководителю |
