⚠️  ВАЖНО: СРАЗУ ПОСЛЕ ПЕРВОГО ВХОДА!

1. Сгенерировать новый пароль:
   node generate-password.js "ВашНовыйСуперПароль123"

2. Скопировать вывод в .env файл:
   nano .env
   (Заменить строку ADMIN_PASSWORD_HASH)

3. Перезапустить сервер:
   Ctrl+C (остановить)
   npm start (запустить снова)
