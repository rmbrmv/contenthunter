## 🚫 Что НЕ коммитить в Git

Файлы, которые не должны попасть в Git (настроено в `.gitignore`):

- `.env` - секретные credentials
- `logs/` - логи работы
- `venv/` - виртуальное окружение
- `__pycache__/` - Python кеш

## ✅ Что коммитить в Git

Безопасно коммитить:

- Весь код (`*.py`)
- Документацию (`*.md`)
- Скрипты (`*.sh`)
- `.env.example` (без реальных credentials)
- `requirements.txt`
- `.gitignore`
