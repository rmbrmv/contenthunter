# 📚 Примеры использования бота

## Для клиентов

### Отправка видео материала

**Сценарий:** Клиент хочет отправить исходное видео для монтажа

```
[Пользователь загружает видео]
Caption: "Исходное видео для корпоративного ролика"

✅ Материал сохранён!
Тип: video
Описание: Исходное видео для корпоративного ролика
```

В Airtable автоматически создаётся запись:
- Date: 2024-02-23 15:30:00
- Client: Иван Петров
- Chat ID: -1001234567890
- Type: video
- Description: Исходное видео для корпоративного ролика
- Telegram File ID: BAACAgIAAxkBAAI...
- Status: new

---

### Вопрос о процессе

**Сценарий:** Клиент спрашивает про сроки

```
@ch_client_service_bot Сколько времени займёт монтаж?

🤖 Обычно монтаж видео занимает 2-5 рабочих дней после получения всех материалов и согласования брифа.

Для более точной оценки нужно знать:
• Длительность финального видео
• Сложность монтажа (простая склейка или сложная обработка)
• Нужна ли цветокоррекция, графика, анимация

Вы можете отправить материалы, и мы дадим точный срок! 🙂
```

## Для менеджеров

### Проверка новых материалов

**Сценарий:** Менеджер проверяет, что пришло за день

```
/search

🔍 Найдено материалов: 15

[Список последних материалов]
```

Или в Airtable: фильтр `Status = new`

---

### Помощь клиенту с процессом

**Сценарий:** Клиент спрашивает про правки

```
@bot Клиент спрашивает, сколько правок включено?

🤖 Стандартно в проект включено 2 итерации правок:

1️⃣ Первая итерация: после показа черновой версии
2️⃣ Вторая итерация: финальные доработки

Дополнительные правки обсуждаются отдельно и могут быть добавлены за дополнительную плату.

При внесении правок клиенту рекомендуется:
• Собрать все замечания в одно сообщение
• Описать максимально конкретно, что нужно изменить
• По возможности приложить референсы

Это ускорит процесс! 🚀
```

---

### Поиск по конкретному клиенту

В Airtable:
- Фильтр: `Client = "Компания ABC"`
- Сортировка: по дате (новые сверху)
- Просмотр всех материалов от клиента

---

### Обновление статуса

**В Airtable вручную:**
1. Открыть запись
2. Изменить `Status`: new → processing → done
3. Добавить `Notes`: "Передано в работу дизайнеру"

---

### Быстрый FAQ

**Сценарий:** Менеджер спрашивает бота о процессе

```
@bot Как работает оплата?

🤖 Процесс оплаты:

1. После согласования брифа и стоимости выставляется счёт
2. Работа начинается после поступления предоплаты (обычно 50%)
3. Окончательная оплата — после утверждения финального результата

Способы оплаты:
• Банковский перевод для юридических лиц
• Онлайн-оплата для физических лиц

Для крупных проектов возможна поэтапная оплата — обсуждается индивидуально.
```

## Типичные кейсы

### Кейс 1: Новый проект с нуля

1. **Клиент:** Отправляет запрос
   ```
   Привет! Нужно сделать видеоролик для YouTube
   ```

2. **Менеджер:** Уточняет детали
   ```
   Здравствуйте! Расскажите подробнее:
   - Какая тематика ролика?
   - Длительность?
   - Есть ли исходные материалы?
   ```

3. **Клиент:** Отправляет бриф + материалы
   ```
   [Текст с описанием]
   [Загружает видео]
   [Отправляет ссылку на Drive]
   ```

4. **Бот:** Сохраняет всё автоматически
   ```
   ✅ Материал сохранён! (× 3)
   ```

5. **Менеджер:** Проверяет Airtable → Все материалы на месте → Согласовывает бриф

---

### Кейс 2: Срочный проект

1. **Клиент:**
   ```
   СРОЧНО! Нужен ролик к завтрашнему дню!
   ```

2. **Бот:** (если настроена детекция ключевого слова)
   ```
   ⚠️ Обнаружен срочный запрос. Менеджер уведомлён.
   ```

3. **Менеджер:** Быстро реагирует
   ```
   Понял, обрабатываем как срочный. Сейчас оценю...
   ```

---

### Кейс 3: Правки к проекту

1. **Клиент:**
   ```
   Посмотрел черновик. Нужны правки:
   - Убрать 3-ю сцену
   - Изменить цветокоррекцию на более тёплую
   - Добавить логотип в конце

   материал: правки к проекту
   ```

2. **Бот:** Детектирует ключевое слово "материал"
   ```
   ✅ Материал сохранён!
   Тип: text
   Описание: Правки к проекту...
   ```

3. **Менеджер:** Видит в Airtable новую запись с типом "text" и ключевым словом "правки" → Передаёт команде

---

### Кейс 4: Вопрос клиента

1. **Клиент:**
   ```
   @bot Как отправить большие файлы?
   ```

2. **Бот:**
   ```
   🤖 Для больших файлов (более 100 MB) рекомендую использовать облачные сервисы:

   📦 Поддерживаемые сервисы:
   • Google Drive
   • Dropbox
   • WeTransfer
   • Яндекс.Диск
   • OneDrive

   Просто отправьте ссылку на файл, и я автоматически сохраню её в нашей базе! 🙂
   ```

---

### Кейс 5: Постоянный клиент

1. **Клиент:** (уже работал раньше)
   ```
   Привет! Снова нужна ваша помощь. Вот материалы:
   [Загружает файлы]
   ```

2. **Бот:** Сохраняет
   ```
   ✅ Материал сохранён!
   ```

3. **Менеджер:** Видит историю клиента в Airtable
   - Фильтр: `Client = "Компания XYZ"`
   - Видит прошлые проекты
   - Понимает контекст и предпочтения

## Заключение

Бот значительно упрощает работу с клиентами:

✅ **Для клиентов:**
- Простота отправки материалов
- Быстрые ответы на вопросы
- Прозрачность процесса

✅ **Для менеджеров:**
- Автоматическая фиксация материалов
- Централизованная база в Airtable
- Меньше рутинных вопросов (отвечает бот)

✅ **Для команды:**
- Все материалы в одном месте
- Понятная структура
- История по каждому клиенту
