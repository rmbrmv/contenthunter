# Changelog

Все значимые изменения проекта будут документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2024-02-23

### Добавлено
- ✅ Telegram бот с поддержкой групповых и личных чатов
- ✅ Автоматическая детекция материалов:
  - Файлы (видео, фото, документы)
  - Ссылки на облачные хранилища (Drive, Dropbox, WeTransfer и др.)
  - Текстовые описания с ключевыми словами
- ✅ Интеграция с Airtable для сохранения материалов
- ✅ AI-ассистент для ответов на вопросы (OpenAI / Anthropic Claude)
- ✅ База знаний (FAQ, процессы работы)
- ✅ Команды бота:
  - `/start` - приветствие
  - `/help` - справка
  - `/search` - поиск материалов
  - `/status` - статистика
- ✅ Валидация файлов (размер, расширения)
- ✅ Allowlist чатов для безопасности
- ✅ Rate limiting
- ✅ Подробное логирование (loguru)
- ✅ Обработка ошибок и graceful shutdown
- ✅ Скрипты для установки и запуска
- ✅ Подробная документация (README, SETUP_GUIDE, USAGE_EXAMPLES)

### Документация
- README.md - основная документация
- SETUP_GUIDE.md - пошаговая инструкция по настройке
- USAGE_EXAMPLES.md - примеры использования
- FAQ в knowledge base
- Описание процессов работы

### Конфигурация
- .env.example - шаблон настроек
- config.py - централизованная конфигурация
- Feature flags для включения/выключения функций
- Поддержка нескольких AI провайдеров

### Скрипты
- setup.sh - автоматическая установка
- run.sh - запуск бота
- check_config.py - проверка конфигурации

### Безопасность
- Allowlist чатов
- Валидация типов и размеров файлов
- Rate limiting
- Безопасное хранение credentials в .env
- .gitignore для защиты секретов
