# 👋 Начните отсюда!

## Что это за проект?

**Client Service Bot** - Telegram бот для автоматизации клиентского сервиса продакшн-студии.

Бот автоматически:
- 📥 Сохраняет материалы от клиентов (файлы, ссылки) в Airtable
- 🤖 Отвечает на вопросы с помощью AI (Claude/OpenAI)
- 🔍 Помогает искать материалы в базе

## 📞 Нужна помощь?

**При проблемах с установкой:**
1. Проверьте `.env` файл
2. Запустите `python check_config.py`
3. Смотрите логи: `logs/bot.log`
4. Читайте FAQ в `knowledge/faq.md`

**При ошибках:**
- Смотрите раздел "FAQ" в README.md
- Проверяйте логи в `logs/bot.log`
- Используйте `check_config.py` для диагностики
