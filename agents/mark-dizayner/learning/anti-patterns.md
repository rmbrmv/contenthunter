## [2026-03-02] SVG аватары вручную | Роман остановил: "Не рисуй SVG руками" | При наличии image API — всегда генерировать через него. SVG/PIL вручную только если API недоступен

## [2026-03-02] Стилистическая несогласованность аватаров | Часть вышла мультяшной, часть полуреалистичной — перегенерация | Для серии аватаров — начинать промпт с одинакового стилистического якоря. Якорь идёт первым, персонаж — вторым

## [2026-03-02] Переусложнение зонирования (10 зон в одном промпте) | "Всё съехало как соты", несвязная композиция | Не больше 4-5 зон в одном промпте. Ограничения важнее перечислений ("единый открытый этаж" > "10 отдельных зон")

## [2026-03-02] Кириллица в AI-промпте как часть изображения | "Варененка", "орботают" — нечитаемо | Никогда не рассчитывать на AI-генерацию читаемой кириллицы. Всё — программно поверх (Pillow, canvas, CSS overlay)

## [2026-03-02] Размер 1792x1024 не проверен перед отправкой | 3 попытки → HTTP 400 | Максимум gpt-image-1 = 1536x1024. Любой размер из задачи — сначала проверить поддержку, при несоответствии — сразу сообщить

## [2026-03-02] Пол агентов без явного указания | Мужчины в женственном стиле, все в костюмах | "MAN"/"WOMAN" в начале описания обязательно. Для нестандартной одежды — явный запрет: "NO suit", "NO tie"

## [2026-03-03] Игнорирование технических ограничений API при пересылке задачи | 3 попытки с неверным размером потрачены | Любой параметр из задачи — сначала проверить поддержку. Не молчать о расхождении

## [2026-03-06] 19 итераций фона без стратегии | ~2 часа потрачено | Лимит попыток одним промптом/моделью = 3. После — сменить модель или подход. Если 3 модели не дали результат — пересмотреть постановку

## [2026-03-06] Показывать варианты без предварительного референса | 7 концептов отвергнуты подряд | Перед генерацией спросить: "Есть референс?" Если нет — предложить 2-3 известных продукта, получить подтверждение, потом генерировать

## [2026-03-06] Спрайты без уточнения требований к анимации | "Плохо двигаются" (3 кадра мало) | Перед генерацией спрайтов — показать образец и спросить "такое ок?". Готовые спрайты с itch.io лучше AI-генерации для walking cycle

## [2026-03-06] Нет fallback при исчерпании API кредитов | LaoZhang дал 401 в середине работы | Перед серийной генерацией (10+ изображений) — тестовый запрос. Иметь готовый fallback (Pillow/SVG) для 401/429

## [2026-03-08] Циклиться на своих вариантах при серии отказов | 9 концептов отклонены, Роман: "ты каждый раз только наши же варианты скидываешь" | Правило трёх отказов: если три концепта подряд отклонены → стоп. Найти реальные продукты как референсы

## [2026-03-17] Ожидание ответа без эскалации | Концепт I висит без ответа 9+ дней, никаких действий | Если задача ожидает ответа >3 рабочих дней — отправить follow-up. Не ждать пассивно

## [2026-03-20] Workspace сброшен — memory/ и learning/ утеряны | Предыдущие записи (02–19 марта) исчезли после пересоздания workspace | Критические файлы (memory/, learning/) должны коммититься в git сразу после создания. Пустой git-репозиторий = нет защиты от потери данных

## [2026-03-23] Знаю правило — не применяю | Правило эскалации зафиксировано 17.03, Концепт I без ответа 15 дней — follow-up так и не отправлен | Знание паттерна без действия = ноль. При фиксации правила в anti-patterns — сразу проверить, применимо ли оно прямо сейчас. Если да — применить немедленно, не откладывать на "следующий контакт"
