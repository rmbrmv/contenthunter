# Approved Patterns — Что сработало

## [2026-03-02] Выбор стиля через 3 концепта | Роман выбрал изометрию ("разъёбище") | Предлагать 3 разных стиля сразу — заказчик выбирает интуитивно, не объясняя словами

## [2026-03-02] Programmatic overlay поверх AI-арта | Текст/имена через Pillow = всегда читаемо | AI-арт = фон/атмосфера; текст, кириллица, UI-данные = canvas/Pillow overlay. Никогда не доверять AI кириллицу

## [2026-03-02] Явное указание пола/возраста/одежды в промпте аватара | 19/19 правильных результатов | Шаблон: "[GENDER], [AGE] years old, [FEATURE]. [CLOTHING], NO [запрет]. Flat vector portrait, solid [COLOR] background"

## [2026-03-02] Откат к лучшей версии при провале итерации | v4 лучше v3 после отката к v2 | Не патчить сломанное — взять предыдущую рабочую версию и сделать минимальное изменение

## [2026-03-02] Проверка API-параметров перед генерацией | Нашёл правильный размер 1536x1024 за 1 тест | При неизвестном API — сначала тест с минимальным запросом, прочитать ответ ошибки

## [2026-03-03] Предлагать 3 варианта → заказчик выбирает интуитивно | Роман выбрал без долгих обсуждений | Для любой визуальной задачи без чёткого ТЗ — 3 концепта в разных стилях одним запросом

## [2026-03-06] ONE SHOT close-up view для широких сцен | Финальный фон 1536×1024 за один запрос, без стыков | Для горизонтальных сцен добавлять: "close-up interior view, fills entire frame edge to edge, no borders"

## [2026-03-06] Pillow overlay для ночного режима | Ночная версия = дневной AI-фон + затемнение + световые пятна за 10 минут | День/ночь = один AI-рендер + Pillow transform. Не генерировать два отдельных варианта

## [2026-03-06] Depth sort по Y для изометрических спрайтов | Ближние персонажи корректно перекрывают дальних | В изометрии с несколькими объектами — depth sort по Y обязателен

## [2026-03-08] Внешние референсы разблокируют диалог при серии отказов | После 9 концептов — показал Habbo и Bramble, Роман выбрал за 2 минуты | Если заказчик отверг 3+ варианта подряд → найти 3-5 реальных продуктов в нише, показать их, спросить "на что из этого похоже?"
