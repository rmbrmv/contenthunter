## [10:52 UTC] Autowarm — хранение паролей соцсетей

### Таблица `social_credentials`
```sql
CREATE TABLE social_credentials (
  id SERIAL PRIMARY KEY,
  username TEXT NOT NULL,
  platform TEXT NOT NULL,  -- instagram | tiktok | youtube | gmail
  password TEXT,
  notes TEXT,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW(),
  UNIQUE(username, platform)
);
```

### API (server.js)
- `GET /api/social/credentials?username=X&platform=Y` → возвращает password + notes
- `POST /api/social/credentials` → upsert (username, platform, password, notes)
- `GET /api/social/credentials/bulk?platform=instagram` → объект {username: {password, notes}} для всей таблицы

### UI (в таблице аккаунтов IG/TikTok/YT)
- Колонка "Пароль": показывает `••••••` если есть, `—` если нет
