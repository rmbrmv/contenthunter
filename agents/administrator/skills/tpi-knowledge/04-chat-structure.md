# Структура рабочего чата Elena-HR

> Система общается с руководителем через структурированный Telegram-чат с топиками. Каждый топик — отдельный канал с чётким назначением.

---

## Топик: General

**Назначение:** Системные сообщения, технические статусы, уведомления о работе системы.

**Активные агенты:** Администратор

**Кто пишет:** Только агенты системы. Руководитель не пишет в этот топик.

**Частота:** При каждом значимом системном событии.

**Примеры сообщений:**

```
🟢 [Elena HR] Система запущена
Период: Неделя 11 (10–16 марта 2026)
Источники данных: Telegram ✅ | Zoom ✅ | Отчёты ✅ | CRM ❌
Активных сотрудников: 8
Расчёт TPI начнётся в понедельник 08:00
```

```
⚠️ [Администратор] Предупреждение о качестве данных
CRM не передавал данные 7 дней.
Метрика «Результативность» будет рассчитана только по отчётам сотрудников.
Confidence для этой метрики: ~0.55
```

```
✅ [Администратор] Еженедельный расчёт завершён
Обработано событий: 2 847
Сотрудников оценено: 8 из 8
Среднее confidence: 0.84
Дашборд доступен в топике «Моя команда»
```

**Правила:**
- Сообщения короткие, технические
- Используются emoji-статусы: 🟢 ok / ⚠️ warning / 🔴 critical
- Никаких персональных данных сотрудников

---

## Топик: Моя команда

**Назначение:** Еженедельный дашборд, командный TPI, аналитические выводы, рекомендации.

**Активные агенты:** HR-директор (основной), Аналитик, Коуч

**Кто пишет:** Агенты — еженедельно; руководитель — обсуждение выводов, уточняющие вопросы.

**Частота:** Раз в неделю (понедельник) + по запросу руководителя.

**Пример сообщения (еженедельный дашборд):**

```
📊 Команда · Неделя 11 · 10–16 марта

Командный TPI: 7.2 / 10  (+0.4 ▲)
Потенциал команды: ≈ ₽2.1M

─────────────────────────────
Рейтинг команды:

🟢 Жукова Т.     8.9  (+0.1)
🟢 Андреева М.   8.4  (+0.6) ⭐
🟢 Борисов К.    7.8  (+0.2)
🟡 Дмитриева Н.  7.1  (+0.3)
🟡 Волкова С.    6.5  (0.0)
🟠 Григорьев А.  5.2  (-2.9) ⚠️
🟠 Ефимов П.     4.8  (-1.2)  ⚠️ данных мало
🔴 Захаров И.    3.4  (-2.1) 🚨

─────────────────────────────
Топ-3 инсайта:
1. Команда растёт 4 недели подряд
2. Григорьев А. — резкое падение, нетипичное для него
3. Захаров И. — изолирован в графе коммуникаций

Подробно → топик «Вопрос/ответ»
```

**Правила:**
- Один дашборд в понедельник до 09:00
- Таблица сотрудников — обязательный блок
- Детальный анализ — только по запросу руководителя
- Имена сотрудников — видны только владельцу

## Топик: 911

**Назначение:** Экстренные сигналы, требующие немедленного внимания руководителя.

**Активные агенты:** Все агенты (сигнал инициирует Data Engine)

**Кто пишет:** Только система. Руководитель читает и действует.

**Частота:** При срабатывании триггера — в любое время.

**Триггеры для алерта:**
| Тип | Порог | Приоритет |
|-----|-------|-----------|
| `SHARP_DROP` | TPI упал на 3+ за неделю | 🔴 High |
| `TOXIC_CONFLICT` | Агрессия/мат в адрес коллеги | 🔴 High |
| `SILENT_DAY` | 2+ рабочих дня без связи | 🟠 Medium |
| `ZERO_PERFORMANCE` | Факт = 0 при наличии плана | 🟠 Medium |
| `CRITICAL_TPI` | TPI < 3.0 | 🔴 High |

**Пример алерта:**

```
🚨 ЭКСТРЕННЫЙ СИГНАЛ · Elena HR

Сотрудник: Григорьев А.
Тип: SHARP_DROP
Когда: 16 марта 2026

Что произошло:
TPI упал с 8.1 до 5.2 за одну неделю (-2.9).
Все 5 метрик снизились одновременно — нетипичный паттерн.

Что это значит:
Синхронное падение всех метрик — признак внешнего стресс-фактора
(не рабочая проблема, а что-то личное или конфликт).

Рекомендуемое действие:
Проведите 1-on-1 сегодня или завтра.
Скрипт: «Алексей, как ты? Есть что-то, что мешает тебе работать в привычном ритме?»

Confidence: 0.85
```

**Правила:**
- Сообщение приходит немедленно при срабатывании триггера
- Не более 1 алерта на сотрудника в день (дедупликация)
- Формат: кто → что → что это значит → что делать
- После того как руководитель отметил алерт решённым — статус меняется

## Топик: Вопрос / ответ

**Назначение:** Диалог руководителя с системой. Детальный анализ по запросу.

**Активные агенты:** HR-директор (роутит вопрос к нужному агенту)

**Кто пишет:** Руководитель задаёт вопросы; агенты отвечают.

**Частота:** По запросу.

**Форматы запросов:**

| Тип запроса | Пример | Агент |
|---|---|---|
| Детальный профиль | «Расскажи подробнее по Андреевой» | Аналитик |
| Социодинамика | «Кто неформальный лидер в команде?» | Социолог |
| Подготовка к встрече | «Готовь briefing к 1-on-1 с Григорьевым» | Коуч |
| Технический статус | «Почему Ефимов получил низкий confidence?» | Администратор |
| Кадровое решение | «Как оценивается динамика Захарова за месяц?» | Аналитик |

**Пример диалога:**

```
👤 Руководитель:
Объясни подробнее по Григорьеву. Что произошло?

🤖 Elena HR → Аналитик:
Григорьев А. · Анализ · Неделя 11

TPI: 8.1 → 5.2 (−2.9 за неделю)

Метрики:
  Коммуникация:   7.8 → 4.5  (−3.3)
  Дисциплина:     8.3 → 5.1  (−3.2)
  Вовлечённость:  7.9 → 4.8  (−3.1)
  Экспертность:   8.5 → 6.0  (−2.5)
  Результативность: 7.9 → 5.1 (−2.8)

Паттерн: все 5 метрик упали синхронно.
Это нетипично для рабочих проблем — там обычно падает 1-2 метрики.
Синхронное падение = внешний фактор (личное, конфликт, выгорание).

Последние 4 недели: 8.3 → 8.1 → 8.0 → 5.2
Предыдущие недели были стабильны.

Confidence: 0.85 ✅
```

**Правила:**
- HR-директор роутит вопрос к нужному агенту автоматически
- Ответ приходит в тот же топик, в ответ на сообщение руководителя
- Формат ответа: структурированный, с данными, без лишних вступлений
- Если вопрос требует нескольких агентов — HR-директор синтезирует ответ

## Права доступа

| Роль | General | Моя команда | 911 | Вопрос/ответ |
|------|---------|-------------|-----|--------------|
| **Owner** (владелец) | Чтение | Чтение + запись | Чтение | Чтение + запись |
| **HR** | Чтение | Чтение + запись | Чтение | Чтение + запись |
| **Team Lead** | ❌ | Чтение (только своя команда) | Чтение (только своя) | Ограниченный запрос |
| **Viewer** | ❌ | ❌ | ❌ | ❌ |

## Форматы сообщений

### Формат: Дашборд недели
```
📊 [Заголовок] · Неделя N · Даты

[Командный TPI] + динамика + деньги
─────────
[Таблица сотрудников]
─────────
[Топ инсайты]
[Алерты если есть]
```

### Формат: Алерт 911
```
🚨/⚠️ ТИП · Elena HR
Сотрудник: [Имя]
Что: [факт]
Что значит: [интерпретация]
Действие: [конкретно]
Confidence: [число]
```

### Формат: Ответ на вопрос
```
🤖 Elena HR → [Агент]:
[Имя сотрудника] · [Тип анализа] · [Период]

[Структурированные данные]
[Вывод]
Confidence: [число]
```
