# Шаблон комментария к тренировке в TrainingPeaks

## Порядок блоков (строго):
1. Комментарий по тренировке
2. 💧 Во время тренировки (вода, изотоник, гели) + КБЖУ
3. 🛏 Сон: Xч Xм (отбой–подъём) | Оценка: X/10
4. 🥗 Завтрак: [еда] → Б/Ж/У = ккал
5. 🥗 [До/После тренировки]: [еда] → Б/Ж/У = ккал
6. 📊 Итого за день: Б/Ж/У = ккал

RPE и Ощущение — в отдельные поля ТП.

## Источники данных:
- Сон → /root/.openclaw/workspace-health-coord/data/sleep.md
- Питание → топик Питание (нутрициолог), /root/.openclaw/shared/data/health/YYYY-MM-DD/nutrition.md
- Вода/изотоник → топик Питание
- Тренировка → ТП (после синхронизации с Garmin)
