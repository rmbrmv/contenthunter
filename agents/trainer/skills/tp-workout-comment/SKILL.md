# Скилл: Комментарий к тренировке в TrainingPeaks

## Рабочий процесс (согласовано с Романом)

1. **Роман:** «тренировку закончил»
2. **Я:** собираю данные (ТП + сон + питание) и присылаю готовый шаблон
3. **Роман:** дописывает свой комментарий и RPE/ощущение прямо в шаблон, отправляет мне
4. **Я:** выставляю RPE и Feeling через API в ТП, отправляю Роману финальный текст
5. **Роман:** копирует текст в Post-activity Comments в ТП вручную

> Post-activity Comments недоступны через API (бесплатный тариф ТП).
> RPE и Feeling — через API автоматически.


## Когда использовать
Роман сообщил что сделал тренировку → автоматически заполнить комментарий, RPE и Ощущение в TrainingPeaks.

## Источники данных

| Данные | Откуда брать |
|--------|-------------|
| Тренировка (TSS, ЧСС, время, дистанция) | ТП API — workout за сегодня |
| Начало тренировки | `workoutDay` + время старта из ТП |
| Сон | `/root/.openclaw/workspace-health-coord/data/sleep.md` |
| Питание (КБЖУ, время) | Сессия нутрициолога `topic:5` или `/root/.openclaw/shared/data/health/YYYY-MM-DD/nutrition.md` |
| Вода/изотоник во время тренировки | Сессия нутрициолога `topic:5` (ищи записи в промежутке тренировки) |
| Комментарий | От Романа (что сказал после тренировки) |

## Порядок блоков (СТРОГО)

```
[Комментарий Романа по тренировке]

💧 Во время тренировки:
- [Напиток/гель объём] → Б0 / Ж0 / УXX = XXX ккал

🛏 Сон: Xч Xм (ЧЧ:ММ–ЧЧ:ММ) | Оценка: X/10

🥗 Завтрак:
- [Блюда] → БXX / ЖXX / УXX = XXX ккал

🥗 До тренировки (ЧЧ:ММ, за ~Xч XXм до старта):
- [Блюда] → БXX / ЖXX / УXX = XXX ккал

📊 Итого за день: БXX / ЖXX / УXX = XXXX ккал
```

RPE и Ощущение — в **отдельные поля** ТП (не в комментарий).

## API ТП

```js
// Получить тренировку за дату
GET https://tpapi.trainingpeaks.com/fitness/v6/athletes/3311233/workouts/YYYY-MM-DD/YYYY-MM-DD

// Обновить тренировку
PUT https://tpapi.trainingpeaks.com/fitness/v6/athletes/3311233/workouts/{workoutId}
Body: { ...workout, description: comment, feeling: X, rpe: X }
```

Авторизация — через Puppeteer (логин rmbrmv, пароль из `.env`).
Скрипт логина: `/root/.openclaw/workspace-health-coord/scripts/tp_fetch.js`
