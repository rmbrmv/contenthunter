## Задача: Массовый импорт стоп-слов в validator (Даниил)

**Запрос:** реализовать механику импорта стоп-слов в раздел Модерация через CSV-файл или Google Sheets.

**Требования:**
- Заголовки файла могут не совпадать с полями БД → маппинг на UI
- Каких-то данных может не быть → дефолты / пустые значения (category — пустой)
- Дубликаты должны отсеиваться автоматически

**Анализ:**
- Таблица: `validator_moderation_rules` (поля: word, category, severity, recommendation, is_active)
- Уникальность: только `word` (case-insensitive) — уже была проверка в `create_mod_rule`
- Код: `workspace-genri/validator/backend` (FastAPI) + `frontend` (Vue 3)

**Что реализовано:**

### Backend (`admin.py`)
- Хелпер `_extract_sheet_id(url)` — извлекает ID из Google Sheets URL регексом
- Хелпер `_parse_csv_bytes(data)` — декодирует UTF-8-sig, парсит через `csv.DictReader`
- Хелпер `_fetch_csv_raw(file, sheets_url)` — загружает CSV из файла или Google Sheets (`export?format=csv`)
- `POST /api/admin/moderation-rules/import/preview` — multipart, параметры: `file`, `sheets_url`, `full` (true/false)
  - `full=false` (дефолт): возвращает первые 10 строк + заголовки + total_rows
  - `full=true`: возвращает все строки (используется при финальном импорте)
- `POST /api/admin/moderation-rules/import/run` — JSON: `{rows, mapping}`
  - Загружает существующие слова из БД в set для быстрой дедупликации
  - Вставляет новые, пропускает дубли (без ошибки), пропускает строки с пустым word
  - Возвращает: `{imported, skipped_duplicates, skipped_errors, duplicate_details, error_details}`

### Frontend
- `ImportModerationModal.vue` — новый компонент
  - Шаг 1: выбор источника (вкладки: файл / Google Sheets)
  - Шаг 2: маппинг колонок + автодетект по похожим названиям + превью 5 строк
  - Шаг 3: результат (карточки: добавлено / дубли / ошибки) + раскрывающиеся детали
- `admin.ts` — добавлены `importModerationPreview` и `importModerationRun`
- `ModerationRules.vue` — кнопка `⬆ Импорт`, перезагрузка таблицы после импорта
- HelpDrawer обновлён — добавлен раздел про импорт
