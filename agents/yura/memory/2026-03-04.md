## [21:00-21:16 UTC] WhatsApp батч-регистрация — дебаг и фиксы

### Проблемы найдены и исправлены:

1. **Диалог выбора магазина** — при открытии `market://details?id=com.whatsapp` Samsung показывает диалог "Galaxy Store vs Google Play". Скрипт не нажимал "Google Play" → зависал.
   - Фикс: добавили `tap_text(xml, 'Google Play', 'Play Маркет')` после открытия Play Store

2. **WA не закрывался после установки** — после install WA сам открывался, `authorize()` пытался запустить его снова → конфликт состояний.
   - Фикс: добавили `am force-stop com.whatsapp` сразу после установки

3. **SMS не приходили** — таймаут ожидания был 120 сек, слишком мало для ALTEL KZ.
   - Фикс: увеличили до 300 сек

4. **Неправильный формат номеров ферма 1** — в Google Sheets номера записаны без ведущей 7: `7082680658` вместо `77082680658`. При загрузке стало `+7082680658` (11 символов) вместо `+77082680658` (12 символов).
   - Фикс SQL: `UPDATE sim_cards SET phone_number = '+7' || SUBSTRING(phone_number FROM 2) WHERE LENGTH(phone_number) = 11 AND phone_number LIKE '+708%';` — исправлено 19 номеров
   - После фикса все 172 номера имеют длину 12 символов (формат +77XXXXXXXXX)
