## Задача: Дедупликация выкладки (Даниил, autowarm)

**Запрос:** ограничение повторной публикации одного видео — не по паку целиком, а по конкретному аккаунту платформы.

**Правило:** `(content_id, account_username, platform)` — уникальная пара. Одно видео можно публиковать на разных платформах/аккаунтах, но не дважды на один и тот же аккаунт.

**Что изменено в `autowarm/server.js`:**
1. `assignUnicResultsToQueue()` — убран блок дедупликации по pack_id (шаг 6), добавлена проверка внутри цикла по аккаунтам
2. `POST /api/publish/queue` — ручное добавление: проверка заменена на `(content_id, account_username, platform)`, возврат 409 при дубле

**Проверено:** pm2 logs показали корректную работу — YouTube блокируется, Instagram/TikTok проходят.

**Документация:** обновлены README.md и NOTES.md в autowarm, изменения запушены на GitHub.

## Фикс TikTok — поле описания не заполнялось (задача #254)

**Причина:** `tap_element(..., clickable_only=True)` не находил поле ввода описания — TikTok рендерит его через Canvas/WebView, UIAutomator не видит элемент как `clickable=true`.

**Фикс (`publisher.py`, метод `publish_tiktok`, Шаг 4):**
- Убрано `clickable_only=True` → поиск без фильтра
- Добавлен fallback-тап по координатам `(540, 290)` если элемент не найден
- pm2 restart autowarm выполнен 2026-03-20

## Фикс autowarm — YouTube title/description (от Genri, 2026-03-20)

**Проблема:** при ручном добавлении задачи YouTube через UI, поле "название" заполнялось из `description` контента вместо `title`, описание оставалось пустым.

**Что изменено (GenGo2/delivery-contenthunter):**

1. `server.js — POST /api/publish/queue/manual` — при `unic_result_id` автоматически подтягивает `title`, `description`, `hashtags`, `geo` из `validator_content` через цепочку `unic_result → unic_task → validator_content`.
   - YouTube: caption = title (≤100 символов), content_description = description, hashtags передаются отдельно
   - Instagram/TikTok: caption = description + hashtags строкой

2. `publisher.py` — убран fallback `caption → description` для YouTube. Описание = только description + hashtags, caption = title.

3. Обновлены README.md, NOTES.md и справка HELP_CONTENT в UI.
