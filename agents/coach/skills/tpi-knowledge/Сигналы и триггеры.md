# Сигналы и триггеры Team Power Index

> Конкретные паттерны поведения которые система читает в данных и переводит в оценку.
> Источники: корпоративные чаты, записи зумов, ежедневные отчёты, CRM.

---

## БЛОК 1 — ИНДЕКС КОМАНДНОЙ НАДЁЖНОСТИ

### 1.1 Коммуникация — сигналы

#### 🔴 Негативные триггеры (снижают оценку)
| Сигнал | Где ловим | Влияние |
|--------|-----------|---------|
| Мат как обращение к коллеге | Чаты | Сильное |
| Жёсткая ирония / язвительность | Чаты, зумы | Среднее |
| Агрессивный тон при получении задачи | Чаты | Среднее |
| Нарушение субординации (публично) | Чаты, зумы | Сильное |
| Грубость при тегах руководителя | Чаты | Сильное |
| Односложные ответы без уважения ("ок", "да", "нет" без контекста при конфликте) | Чаты | Слабое |
| Высказывание неверия / пессимизма по задаче | Чаты, зумы | Среднее |

#### 🟢 Позитивные триггеры (повышают оценку)
| Сигнал | Где ловим | Влияние |
|--------|-----------|---------|
| Благодарность коллегам | Чаты | Среднее |
| Конструктивная критика (предлагает решение) | Чаты, зумы | Среднее |
| Поддержка коллеги в сложной ситуации | Чаты | Среднее |
| Уважительный тон даже в конфликте | Чаты, зумы | Сильное |
| Позитивные реакции, поддержка команды | Чаты | Слабое |

---

### 1.2 Дисциплина — сигналы

#### 🔴 Негативные триггеры
| Сигнал | Где ловим | Влияние |
|--------|-----------|---------|
| Опоздание на зум (упоминание в транскрипте) | Зумы | Среднее |
| Не сдал ежедневный отчёт | Отчёты | Среднее |
| Сдал отчёт позже срока | Отчёты | Слабое |
| Скопировал вчерашний отчёт (одинаковый текст) | Отчёты | Сильное |
| Не ответил на тег в рабочее время >2 часов | Чаты | Среднее |
| Пропал в чате на весь рабочий день (0 сообщений) | Чаты | Среднее |
| Не выполнил поставленную задачу в срок | Таскер | Сильное |
| Передал задачу не по регламенту | Чаты, таскер | Слабое |

#### 🟢 Позитивные триггеры
| Сигнал | Где ловим | Влияние |
|--------|-----------|---------|
| Всегда вовремя на зумах | Зумы | Среднее |
| Предупредил заранее об опоздании/отсутствии | Чаты | Среднее |
| Сдаёт отчёт стабильно в срок | Отчёты | Среднее |
| Отвечает на теги быстро (<30 мин) | Чаты | Слабое |

---

### 1.3 Вовлечённость — сигналы

#### 🔴 Негативные триггеры
| Сигнал | Где ловим | Влияние |
|--------|-----------|---------|
| Похуизм и равнодушие в ответах | Чаты | Среднее |
| Сухость / бинарность (только да/нет без участия) | Чаты | Слабое |
| Высказывание неверия в цели / задачи | Чаты, зумы | Среднее |
| Игнорирует общие обсуждения команды | Чаты | Слабое |
| Ноль инициатив за 2+ недели | Чаты, зумы | Среднее |
| В отчёте — формальная отписка без анализа | Отчёты | Среднее |

#### 🟢 Позитивные триггеры
| Сигнал | Где ловим | Влияние |
|--------|-----------|---------|
| Предлагает идею / улучшение | Чаты, зумы | Сильное |
| Задаёт вопросы по стратегии / целям | Чаты, зумы | Среднее |
| В отчёте — вдумчивый самоанализ, точки роста | Отчёты | Сильное |
| Берёт задачу сверх своей роли | Таскер, чаты | Сильное |
| Интересуется результатами других / команды | Чаты, зумы | Среднее |

## БЛОК 2 — ИНДЕКС РЕЗУЛЬТАТИВНОСТИ

### 2.1 Экспертность — сигналы

#### 🔴 Негативные триггеры
| Сигнал | Где ловим | Влияние |
|--------|-----------|---------|
| Неверный ответ клиенту | Клиентские чаты/зумы | Сильное |
| Не смог обработать возражение клиента | Клиентские чаты/зумы | Сильное |
| Путается в продукте / процессе | Клиентские чаты/зумы | Среднее |
| Пропустил вопрос в отчёте | Отчёты | Среднее |
| Одинаковый ответ на разные вопросы (копипаст) | Отчёты | Сильное |
| Общие фразы вместо конкретики | Отчёты, чаты | Слабое |

#### 🟢 Позитивные триггеры
| Сигнал | Где ловим | Влияние |
|--------|-----------|---------|
| Чёткий развёрнутый ответ клиенту | Клиентские чаты/зумы | Среднее |
| Грамотная обработка возражения | Клиентские чаты/зумы | Сильное |
| Реальный анализ в отчёте (называет конкретные ошибки) | Отчёты | Сильное |
| Делится экспертизой с командой | Чаты | Среднее |
| Называет точки роста с конкретикой | Отчёты | Сильное |

---

### 2.2 Результативность — сигналы

#### 🔴 Негативные триггеры
| Сигнал | Где ловим | Влияние |
|--------|-----------|---------|
| Факт < 70% от плана | CRM, отчёты | Сильное |
| Задача переходит из недели в неделю | Таскер | Сильное |
| CRM не заполнена / пустые поля | CRM | Среднее |
| Не написал отчёт вовремя | Отчёты | Среднее |
| Плановые показатели не выполнены 2+ недели подряд | CRM, отчёты | Критично |

#### 🟢 Позитивные триггеры
| Сигнал | Где ловим | Влияние |
|--------|-----------|---------|
| Факт ≥ 100% от плана | CRM, отчёты | Сильное |
| Все задачи закрыты в срок | Таскер | Среднее |
| CRM заполнена полностью и актуально | CRM | Среднее |
| Перевыполнение 2+ недели подряд | CRM, отчёты | Сильное |

## ЭКСТРЕННЫЕ СИГНАЛЫ
*Требуют немедленного внимания руководителя — экстренный отчёт в тот же день*

| Сигнал | Источник |
|--------|----------|
| Токсичный конфликт в чате (мат + агрессия к коллеге) | Чаты |
| Сотрудник не выходил на связь весь рабочий день | Чаты |
| Факт = 0 при наличии плана | CRM |
| Резкое падение оценки (−3 и более за неделю) | Система |
| Повторный скандал / нарушение субординации | Чаты, зумы |
| Коллега повторно задаёт вопрос который остался без ответа | Чаты |
