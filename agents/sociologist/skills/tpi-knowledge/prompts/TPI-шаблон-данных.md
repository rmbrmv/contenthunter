# TPI — Шаблон подачи данных (User Prompt)

> Это шаблон user prompt — того что подаётся в модель вместе с мастер-промптом.
> Заполняется автоматически скриптом из БД (см. SQL-выборка-данных.md).

---

## USER PROMPT TEMPLATE

```
Оцени сотрудника за период с {date_from} по {date_to}.

## Сотрудник
Имя: {employee_name}
Telegram: @{telegram_username}
Должность: {position}
Отдел: {department}

---

### Внутренние чаты (корпоративные)
Количество сообщений за период: {internal_msg_count}

{internal_messages}

---

### Клиентские чаты
Количество сообщений за период: {client_msg_count}

{client_messages}

---

## БЛОК 2 — ДАННЫЕ ИЗ ZOOM-ТРАНСКРИПТОВ

Встреч за период: {zoom_meetings_count}

{zoom_excerpts}

---

## БЛОК 3 — ЕЖЕДНЕВНЫЕ ОТЧЁТЫ

Отчётов за период: {reports_count} из {working_days} рабочих дней

{daily_reports}

---

## БЛОК 4 — ПЛАН-ФАКТ (если есть)

{plan_fact_data}

---

## ПРЕДЫДУЩАЯ ОЦЕНКА (для сравнения динамики)

```
