# WB API — правила подсчёта

## Заказы (orders)

- Эндпоинт: `statistics-api.wildberries.ru/api/v1/supplier/orders`
- Фильтрация: по полю `date` (дата заказа)
- **Считать ВСЕ заказы** — включая `isCancel: true`
- **Выручка = `priceWithDisc`** (цена после скидки продавца, до SPP)
- **Время = МСК (UTC+3)** — API отдаёт UTC, нужно прибавлять 3 часа
- `finishedPrice` — не использовать для выручки (после SPP, занижает)
- `forPay` — сумма к выплате продавцу (отдельная метрика)

## Сходимость с WB Partners

- Методом выше получаем ±1 заказ и ±1-2% по выручке (погрешность пограничных заказов)
- Этого достаточно для аналитики

## Продажи (sales) — это ВЫКУПЫ, не заказы

- Эндпоинт: `statistics-api.wildberries.ru/api/v1/supplier/sales`
- `totalPrice > 0` — продажа, `totalPrice < 0` — возврат

## Неделя

- Считать пн–вс включительно по МСК
- Диапазон UTC: пн 00:00 МСК = вс 21:00 UTC предыдущего дня
