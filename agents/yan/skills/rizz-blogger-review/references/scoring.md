# Scoring Model (100 pts)

## Формула

| Критерий | Макс. баллов | Детали |
|---|---|---|
| Медиана просмотров | 30 | <500=0, 500-1к=10, 1к-3к=20, ≥3к=30 |
| Content fit (vision) | 25 | 0-25 из vision-анализа |
| ER quality | 20 | <1%=0, 1-3%=10, 3-8%=20, >8%=15 |
| Платформа | 15 | Instagram=10, TikTok=8 |
| Тренд аудитории | 10 | 📈=10, ➡️=5, 📉=0 |
| Бот-штраф | — | 🔴=-20, 🟡=-10, 🟢=0 |

## Рекомендации по итоговому score

- **✅ Принять**: vision=✅ AND score≥55, либо score≥60 (любой verdict)
- **⚠️ Рассмотреть**: score 40-59
- **❌ Отклонить**: score <40

## Content Fit (0-25) — критерии vision-анализа

| CF | Значение |
|---|---|
| 20-25 | Профиль максимально подходит: fashion-фокус, минимализм, ЦА совпадает |
| 14-19 | Частичное совпадение: есть стиль, но не главная тема |
| 8-13 | Смежная тематика (распаковки WB, lifestyle) |
| 0-7 | Нерелевантно: мамский блог, бьюти-услуги, юмор, спорт |
