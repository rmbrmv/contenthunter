## 19:43 — Формат vision-анализа финализирован

### Итоговый формат оценки блогера (4 блока):
1. 🔴/🟡/🟢 **BOT RISK** — like rate, comment rate, follower/following ratio, вирусные выбросы
2. 🔴/🟡/🟢 **CONTENT FIT** — разбор 9 кадров, тематика, стиль одежды в кадре
3. 📊 **Реальный охват** — медиана без вирусных постов (реальный прогноз для рекламы)
4. ✅/⚠️/❌ **ИТОГ** с обоснованием

### Ключевые метрики для bot detection:
- Like rate норма: 3-8% (лайки/просмотры)
- Comment rate норма: 0.1-0.5%
- Вирусные выбросы = один залётный пост, не накрутка — нужно считать медиану без него

### Тестовые разборы показали:
- @savelieva__irina ❌ (like rate 0.66%, контент не про стиль, реальный охват 600-1000)
- @pr.kofanova ❌ (вирус без конверсии, охват 300-600, контент про юмор)
