## 18:19–18:32 UTC — Баги unic-worker: исправление двух критических ошибок

### Баг 1: Валидация длины (worker.py ~line 277)
**Симптом:** "Длительность вне допуска: ожидалось ~7.8s, получилось 3.9s (ratio=0.50)"
**Причина:** validate_output() считала expected = orig_dur / speed, не учитывая video_trim_start
- orig_dur = 9.75s (полный файл), speed=1.25, trim_start=5.0
- Правильно: effective_dur = 9.75 - 5.0 = 4.75; expected = 4.75/1.25 = 3.8s
- Ошибочно: expected = 9.75/1.25 = 7.8s → ratio = 3.9/7.8 = 0.5 → провал
**Фикс:**
```python
trim_start = float(scheme.get('video_trim_start') or 0.0)
effective_dur = max(0.1, orig_dur - trim_start)
expected = effective_dur / speed
```

### Баг 2: S3 upload XAmzContentSHA256Mismatch (worker.py ~line 35)
**Симптом:** "An error occurred (XAmzContentSHA256Mismatch) when calling the PutObject operation"
**Причина:** boto3 1.35.0 включил автоматический checksum по умолчанию; Beget S3 его не поддерживает
**Фикс:** добавить в Config:
```python
config=Config(
    signature_version='s3v4',
    request_checksum_calculation='when_required',
    response_checksum_validation='when_required',
)
```
