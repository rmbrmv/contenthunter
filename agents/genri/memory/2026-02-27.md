# 2026-02-27 — Подключение к ферме телефонов + план разработки autowarm

## Ферма устройств (170 телефонов на малинках)

### Как подключаться к ADB

Устройства подключены к Raspberry Pi, ADB серверы доступны через удалённый хост.
**Не через `adb connect`** — через флаг `-H` (remote ADB server).

```bash
# Список устройств на малинке 1
adb -H 147.45.251.85 -P 15037 devices

# Команда на конкретном устройстве
adb -H 147.45.251.85 -P 15037 -s <SERIAL> shell "<команда>"
```

### Найти на какой малинке конкретный телефон

```bash
for adb_port in 15017 15037 15028 15048 15058 15038 15068 15088 15098 15108; do
  result=$(adb -H 147.45.251.85 -P $adb_port devices 2>&1 | grep "<SERIAL>")
  if [ -n "$result" ]; then echo "Found on port $adb_port: $result"; fi
done
```

### Пример — RF8Y91F8EJH (Samsung SM-A175F)
- Малинка 1, порт **15037**
- Открыть Instagram: `adb -H 147.45.251.85 -P 15037 -s RF8Y91F8EJH shell "monkey -p com.instagram.android -c android.intent.category.LAUNCHER 1"`

### Двойной тап для лайка
```bash
adb -H 147.45.251.85 -P <adb_port> -s <SERIAL> shell "input tap 540 1000; input tap 540 1000"
```
Оба тапа в одной команде — иначе задержка сети слишком большая.

### SSH на малинки
- SSH не работает с нашего сервера (ключ не подходит)
- ADB через `-H` — единственный рабочий способ

## Проект: Autowarm 2.0 — умный прогрев аккаунтов

### Концепция
Полностью новый софт (не на базе autowarm_tasks). Скрипт + ИИ.

### Источники данных
- `distribution.raspberry_port` → ADB подключение к телефону
- `distribution.factory_projects` → проекты
- `distribution.hashtags` → ключевые слова по project_id
- Airtable `Брифы по проектам` (app7fFym01YjkYHB8) → описание, ЦА, ключевые фразы
- Google Sheets `Новый план прогрева` (1RsFVfNTP7aj3fgjNw6Au5fPWM81eKvDbsECcQ0znX-I) → действия по дням

### Стек
- Python (скрипт прогрева)
- PostgreSQL (хранение задач, протоколов)
- Веб-интерфейс на порту 3849

### Алгоритм обработки каждого видео

1. **UI dump** (бесплатно) — парсим текст, описание, хэштеги
   - Найдено "Реклама" / "Sponsored" → пропускаем (свайп без лайка)
   - Текст совпадает с ключевыми словами → лайкаем
   - Текст не дал ответа → скриншот в Haiku

2. **Haiku vision** (~$0.0004/видео) — только если текст не помог
   - Проверяем соответствие списку ключевых слов проекта
   - Совпало → лайкаем / Не совпало → пропускаем

3. **Sonnet** (~$0.003) — только для генерации комментариев (живые, не шаблонные)

### Рандомизация (антибан)
- Паузы между действиями: случайные (не фиксированные)
- Координаты тапа: случайные в зоне центра экрана (x: 400-680, y: 800-1200)
- Скорость свайпа: 600-1200мс
- 20% вероятность пропустить лайк даже если видео подходит
- Иногда не ставить комментарий

### Стоимость
- 1 аккаунт, 10 дней прогрева: **~$0.41**
- 170 устройств полный цикл: **~$70**

### Веб-интерфейс (порт 3849)
- `/protocols` — список протоколов прогрева (по соцсетям и тематикам)
- `/protocol/edit` — редактор по дням: действия, количества, тайминги
- `/tasks` — запущенные задачи (телефон + аккаунт + день + статус)
- `/run` — запустить прогрев на конкретном телефоне

### В будущем — публикация видео
- Видео хранится в S3 (уже с текстом, хэштегами, геотегом)
- `adb push` → телефон → Instagram → автопубликация
- ИИ знает что вставить в описание

### Следующий шаг

## Реальные координаты для ADB (Samsung SM-A175F, 1080x2340)

### Instagram Reels
- Лайк: двойной тап `input tap {x} {y}; input tap {x} {y}` по центру видео (x: 400-680, y: 800-1200)
- Скролл: `input swipe 540 1400 540 400 {600-1200мс}`

### TikTok
- Лайк: двойной тап по центру видео (та же механика что Instagram) ✅
- Скролл: `input swipe 540 1400 540 400 800`
- package: com.zhiliaoapp.musically

### YouTube Shorts
- Открывать через: `am start -a android.intent.action.VIEW -d "https://www.youtube.com/shorts" com.google.android.youtube`
- Лайк: ОДИНОЧНЫЙ тап по кнопке справа
  - bounds: [900,1044][1080,1213] → tap x: 930-1060, y: 1060-1200
  - ВАЖНО: сначала тапнуть центр (540, 1000) чтобы снять паузу, затем лайк
- Скролл: `input swipe 540 1400 540 400 800`
- Детекция лайка: content-desc содержит "Нравится" без "поставить" = лайк активен
- package: com.google.android.youtube
