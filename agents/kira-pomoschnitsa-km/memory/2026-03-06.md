### Методология запросов (выученная)
- **Просмотры за неделю** = SUM(firs.sum_views) WHERE collected_at IN [период]
- **Публикации за неделю** = COUNT(DISTINCT fir.ig_media_id) WHERE fir.timestamp IN [период]
- **Архив** = COUNT(DISTINCT fir.ig_media_id) БЕЗ фильтра по дате (или с фильтром по договорённости)
- **sum_views** = дневной прирост просмотров (не накопительный)
- **views_count** = абсолютное значение на момент сбора
