## Форма еженедельных отчётов

### Текстовый отчёт (Vision форма — ОБЯЗАТЕЛЬНО)
**Файл:** `/root/.openclaw/workspace-kira-pomoschnitsa-km/REPORT_TEMPLATE.md` ← читать перед каждым отчётом!

⚠️ Всегда читать REPORT_TEMPLATE.md перед формированием текстового отчёта клиенту.

Структура (строго по шаблону):
1. `📊 Проект — еженедельный отчёт` + `📅 Даты`
2. `Привет!` + 1–2 честных предложения о неделе
3. `⸻` разделитель
4. `📈 Метрики за неделю` с ▸ (публикации, просмотры, avg, аккаунты, архив)
5. `📍 Накопительный итог` строки
6. `⸻` разделитель
7. Анализ (3–4 абзаца, без заголовка)
8. `⸻` разделитель
9. `🏆 ТОП-10 роликов периода` — нумерованный, ▶️/🎵/📸 + X 👁 ссылка
10. `⸻` разделитель
11. `🚀 Что нужно делать дальше` с ▸

**Язык:** Русский, деловой, честный, без пустой бодрости.

### HTML-дашборд (ПРАВИЛЬНАЯ ФОРМА)
**Файл-шаблон:** `/root/.openclaw/workspace-kira-pomoschnitsa-km/HTML_DASHBOARD_TEMPLATE_V2.html`

**Стиль (по форме Фёдора):**
- Компактный (480px max-width)
- Тёмная тема (#0f0f13 / #1a1a24)
- Фиолетовый акцент (#6366f1)
- Зелёный для роста (#34d399), красный для падения (#ef4444)
- Карточки в grid 2x2 (просмотры, avg, публикации, аккаунты)
- График по неделям (факт vs план) с визуальными барами
- ТОП-10 с относительной высотой барров
- Раздел действий с эмодзи

**Основные компоненты:**
1. **Метрики** — 4 карточки + 1 широкая (архив + накопительный итог)
2. **Прогресс плана** — полоса выполнения + маркеры + план на дату vs факт
3. **График по неделям** — бары (факт синий, план штрих-пунктир)
4. **ТОП-10 роликов** — ранг, платформа (эмодзи), видимый бар, ссылка, просмотры
5. **Действия** — список с эмодзи и текстом

**Плейсхолдеры для замены:**
- `{{ПРОЕКТ}}` — название
- `{{ДАТА_НАЧАЛО}}` / `{{ДАТА_КОНЕЦ}}` — период
- `{{ПРОСМОТРЫ_НЕДЕЛЯ}}`, `{{ПРОСМОТРЫ_ПРОШЛАЯ}}`, `{{ДЕЛЬТА_ПРОСМОТРЫ}}`, `{{КЛАСС_ДЕЛЬТА_ПРОСМОТРЫ}}` (delta-up / delta-down)
- `{{СРЕДНИЙ_ОХВАТ}}`, `{{СРЕДНИЙ_ОХВАТ_ПРОШЛАЯ}}`, `{{ДЕЛЬТА_ОХВАТ}}`, `{{КЛАСС_ДЕЛЬТА_ОХВАТ}}`
- `{{ПУБЛИКАЦИИ}}`, `{{ПУБЛИКАЦИИ_ПРОШЛАЯ}}`
- `{{АКТИВНЫЕ_АККАУНТЫ}}`, `{{ВСЕГО_РОЛИКОВ}}`, `{{ПРОСМОТРЫ_АРХИВА}}`
- `{{НАКОПИТЕЛЬНО_ТЕКУЩЕЕ}}`, `{{ПРОЦЕНТ}}`, `{{ПЛАН_НА_ДАТУ}}`
- `{{PROGRESS_MARKERS}}` — HTML для маркеров (нед. 1, нед. 2 и т.д.)
- `{{CHART_BARS}}` — HTML для барров (по одному bar-wrap на неделю)
- `{{TOP10_ITEMS}}` — HTML для 10 ролик-итемов
- `{{ПЕРИОД_ТОП10}}` — "недели" / "периода" и т.д.
- `{{РАЗДЕЛ_ДЕЙСТВИЙ_ЗАГОЛОВОК}}` — "Что нужно делать" / "Что делаем для ускорения"
- `{{NEXT_ITEMS}}` — HTML для действий

**Как использовать:**
1. Скопируй шаблон и переименуй: `[project]_dashboard_[period].html`
2. Замени все `{{ПЛЕЙСХОЛДЕРЫ}}` на реальные данные
3. Для `{{PROGRESS_MARKERS}}`, `{{CHART_BARS}}`, `{{TOP10_ITEMS}}`, `{{NEXT_ITEMS}}` — генерируй HTML-блоки
4. Отправь Ане в Telegram вторым сообщением (после текстового отчёта)
