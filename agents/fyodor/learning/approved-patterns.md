## 2026-03-02: Поп-ап Instagram через Puppeteer
**Что сработало:** Удаление поп-апа через `page.evaluate(() => document.querySelectorAll('[role="dialog"]').forEach(el => el.remove()))` + `document.body.style.overflow = 'auto'`
**Почему:** JS напрямую убирает DOM-элемент, не зависит от селектора кнопки закрытия
**Паттерн:** Всегда использовать JS-удаление модалок для Instagram, не пытаться кликнуть крестик

**Почему:** Роман принял формат без замечаний по дизайну
**Паттерн:** Использовать этот шаблон для всех еженедельных встреч CH; обновлять только данные

## 2026-03-02: Анализ клиентских чатов через SQL
**Что сработало:** Быстрый срез по message_date + chat_title + фильтр по username команды
**Почему:** telegram_messages хорошо структурирована, запросы быстрые

## 2026-03-02: Вставка строки в Google Sheets через batchUpdate insertDimension
**Что сработало:** insertDimension + последующий values().update() на точный range
**Почему:** Чистый способ добавить строку не ломая формулы
**Паттерн:** Сначала insertDimension, потом update — никогда не писать сразу без вставки строки

## 2026-03-03: Поиск нужной встречи по двум признакам
**Что сработало:** Определил нужную встречу из 10 сегодняшних — проверил preview транскрипта на наличие "Анна" + тематики дашборда. Нашёл за 3-4 итерации.

## 2026-03-03: Чтение русских транскриптов через subprocess + errors='replace'
**Что сработало:** `subprocess.run(..., capture_output=True, text=True, errors='replace')` обходит UTF-8 encoding errors в psql
**Почему:** psql падает на 0xd0/0xd1 (кириллица), но subprocess с errors='replace' просто заменяет битые символы
**Паттерн:** Для content_text из mymeet.meetings ВСЕГДА использовать subprocess shell=True с errors='replace', не psycopg2 и не голый psql

## 2026-03-11: Методология просмотров через sum_views
**Что сработало:** Использование `sum_views` из `factory_inst_reels_stats` для дневных дельт вместо `account_daily_delta` или `views_count`
**Почему:** `sum_views` — уже вычисленные дневные просмотры без артефактов первого снимка; `views_count` — накопительный total
**Паттерн:** Для любых дельт за период: `SUM(sum_views) WHERE collected_at BETWEEN X AND Y`. Для текущего состояния: `DISTINCT ON (ig_media_id) ORDER BY collected_at DESC`

## 2026-03-03: Саммари встречи по сырому транскрипту
**Что сработало:** Прочитал 73К символов транскрипта (3 части по 15К+), вычленил 7+ конкретных правок к отчёту + одобренные метрики
**Почему:** Полное чтение даёт контекст — можно отделить принятые решения от дискуссий
**Паттерн:** Читать транскрипт тремя частями (head / middle / tail), затем синтезировать. Искать глаголы-решения: "давай", "добавь", "убери", "нужно", "окей".
